## What is Web Server?

- It is referred to as servers located in data centers
- It contains the files and webpages for websites
- Using Internet and HTTP protocol, it delivers and displays the files or websites

## Fundamentals of WorldWideWeb

1. HyperText Markup Language (HTML)
2. Uniform Resource Identifier (URI)
3. HyperText Transfer Protocol (HTTP)

## Web Server Software

- It is a software which is installed on a server
  - transforms the server into a web-server
- It can be used to serve
  - static files
  - dynamic files
- The content is served using the HTTP Protocol
